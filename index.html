About this project source code

Well, the code is open source. By looking at it, you will realize that it is a simple Ruby on Rails application. The application presents a single page where you can enter one or more links to github projects (Rails applications). The links are stored in the database and in parallel a demon should be running (delayed job). The purpose of this process is to check for new links on the database, use them to clone the projects repositories, retrieve github details, run ruby_best_practices gem upon them and store everything in the database.

Dependencies

The project consists of a regular Rails application, Its dependencies are listed in the Gemfile. The only uncommon thing is the need to use my altered version of the Richard Huang gem: rails_best_practices
